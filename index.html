ScriptLens Privacy Policy

Last updated: May 2026

ScriptLens is a browser extension that visualizes dependencies between ServiceNow Script Includes and other ServiceNow artifacts. This policy explains what data the extension accesses and how it is used.

Data We Access

ScriptLens accesses the following data solely to provide its core functionality:

  • ServiceNow page content: When you navigate to a Script Include page, the extension reads the script name and script body from the page.
  • ServiceNow session token (g_ck): The extension reads your active session token from the ServiceNow page to authenticate API requests on your behalf. This token is never stored persistently and is only held in memory for the duration of the visualization session.
  • ServiceNow REST API: The extension makes authenticated requests to your ServiceNow instance's REST API to retrieve dependent Script Include records needed to build the dependency graph.

Data We Do Not Collect

  • We do not collect, transmit, or store any data on external servers.
  • We do not track your browsing activity.
  • We do not collect personally identifiable information.
  • We do not use analytics or advertising services.
  • All data accessed by the extension stays within your browser and your ServiceNow instance.

Permissions

ScriptLens requests the following browser permissions:

  • activeTab / tabs: To detect when you are on a ServiceNow Script Include page and to open the visualization tab.
  • scripting: To inject the content script that reads the script body from the ServiceNow page.
  • host_permissions (*://*.service-now.com/*): To make authenticated API requests to your ServiceNow instance to fetch dependent Script Include records.

Data Retention

ScriptLens does not persist any data between browser sessions. All visualization data is held in the extension's background service worker memory and is cleared when the browser or extension is restarted.

Third-Party Services

ScriptLens does not integrate with or send data to any third-party services. The only external communication is between your browser and your own ServiceNow instance.

Changes to This Policy

We may update this privacy policy as the extension's features evolve. The "Last updated" date at the top of this page will reflect any changes. Continued use of the extension after changes are posted constitutes acceptance of the updated policy.

Contact

If you have any questions about this privacy policy or ScriptLens, please open an issue on the ScriptLens GitHub repository.
